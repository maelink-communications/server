# maelink-server

Server for maelink, the open source social media

> [!WARNING]
> This code is soon to be refactored. Please use the more up-to-date `no-more-onefile` branch which has multiple QOL fixes.

## HOW TO RUN

First, install the required packages with:
`deno task install`

Once you have installed the packages you can now run the server with:
`deno task start`
